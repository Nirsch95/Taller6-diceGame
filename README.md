# Ejercicio caso práctico de una página dinámica

Juego de dados para múltiples jugadores.

Proyecto demo en donde se una una separación de responsabilidades usando los conceptos de:

*	Plantilla (vista)
*	Lógica (controlador)
*	Datos (Modelo)
